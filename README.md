## ‚¨áÔ∏è Installation
To get started, follow these steps:

1. Fork the repository
2. Clone the repository or download it
3. Open your terminal and inside the project folder, run :
```bash
  npm install
```
4. Start the dev server
```bash
  npm run dev
```
